# Suicidal-Nights-cpp
The indie horror Game, made on a C++ with raylib.  Survive nights and evade the Entities. READ "HOW_TO_PLAY.txt" BEFORE PLAY! If you need to test Game, download The Portable Version. Portable Version is 2x  less File size than Original. Important announcement for players! Some antivirus programs (such as VirusTotal) may report the game as a "Trojan." This is an error. Why this happened: I compressed the game files using UPX to reduce its size. Unfortunately, viruses also often use UPX, so antivirus programs react alarmingly to any compressed files. The game is completely safe. If you're worried, you can run it in the sandbox or wait for an update where I'll remove this compression. Sorry for the inconvenience! raylib: https://www.raylib.com/ itch.io: https://ch92-creator.itch.io/suicidal-nights
